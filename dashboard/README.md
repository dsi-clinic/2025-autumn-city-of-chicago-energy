# Loading the Streamlit Dashboard

Commands to run in terminal
1. cd src/dashboard
    - Should be in the dashboard folder
2. streamlit run Introduction.py

You might be met with a welcome message in the terminal asking for your email
- You can leave it blank and press ENTER

# Streamlit Dashboard folders and file
- Introduction.py is the main loading file that ties all other pages together and used to run the 
- The pages folder holds all other pages
    - the naming convention is #_{name} with the # being the order of the page in the navigation with the {name} being its title
- dashboard_utils.py is in the utils folder for function that will be used multiple times
